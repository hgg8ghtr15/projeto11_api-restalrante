--------------------------------------------------------------------------------
Iniciar Projeto Novo
--------------------------------------------------------------------------------
Passo a passo para usar/entender o sistema
1-Instalação

Rode npm install (ou yarn) na raiz do projeto para baixar dependências (express, typescript, etc.).

2-Compilação/execução

Se houver script no package.json, use npm run dev ou similar; caso contrário, compile tsc e execute node dist/server.js.
Ao iniciar, o console mostrará Servidor rodando na porta. http://localhost:3333.

3-Testar a rota de produtos

Faça uma requisição GET para http://localhost:3333/products (Postman, curl, navegador).
O retorno atual será:

4- Provocar um erro customizado

No controller há um throw new AppError("Error fetching products"); comentado. Descomente-o e reinicie a aplicação para ver como o middleware trata erros: a resposta será um status 400 com o JSON { "message": "Error fetching products" }.

5- Estrutura de expansão

Para adicionar outras rotas basta criar routers/controlllers semelhantes e montá‑los em index.ts.
Utilize AppError para lançar exceções controladas e elas serão formatadas automaticamente.

6-Desenvolvimento futuro (não implementado ainda, mas previstos)

Implementar lógica de listagem, criação, atualização e exclusão de produtos (possível conexão com banco de dados).
Adicionar validações e middlewares adicionais (autenticação, CORS etc.).
